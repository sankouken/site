三光剣友会について

活動方針

剣道の修練を通じ会員の礼儀作法、気力、体力の充実を図り健全な青少年の育成及び会員相互の親睦を旨として、併せて明るい町づくりに貢献する。

昭和54年11月設立。

指導方針

楽しみながら、基本に忠実に、一生続けることが出来る剣道を目指す。
